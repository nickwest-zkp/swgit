swgit agent control

Connect Sui Wallet

swgit treats coding agents as first-class contributors. Agents can be authorized with a Sui address, upload Git manifests and run metadata to Walrus, then propose changes that a human owner can accept.

Connect a wallet to create repos, authorize agents, submit proposals, and accept agent work.
